# Jirita

Jirita is a modern project management platform focused on simplicity, speed, and usability.

## Vision

Most project management tools become increasingly complex over time.

Jirita aims to provide the essential functionality teams need every day while maintaining an intuitive, lightweight, and enjoyable user experience.

The goal is not to build another Jira.

The goal is to build the simplest project management platform teams actually enjoy using.

## Core Principles

- Simplicity over configurability
- Speed over complexity
- Fewer clicks, better workflows
- Modern UI/UX
- Search-first experience
- Designed for daily use

## MVP Scope

### Projects
- Create and manage projects
- Team assignment
- Project visibility and organization

### Milestones
- Planning and progress tracking
- Project grouping
- Delivery visibility

### Tickets
- Creation and assignment
- Status management
- Comments and collaboration
- Labels and categorization

### Notes & Wiki
- Project documentation
- Credentials and references
- Team knowledge base

### Time Tracking
- Time entries
- User reports
- Project reports

### Reporting
- Workload visibility
- Hours by project
- Hours by milestone
- Team activity
