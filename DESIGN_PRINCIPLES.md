# Design Principles

## 1. Simplicidad extrema

Si una pantalla necesita explicación, está mal diseñada.

## 2. Menos clics

Toda acción frecuente debe tomar máximo 3 clics.

## 3. Interfaces limpias

Evitar tablas gigantes y formularios extensos.

## 4. Velocidad percibida

La aplicación debe sentirse instantánea.

## 5. Mobile Friendly

Toda funcionalidad debe funcionar correctamente desde móvil.

## 6. Progressive Disclosure

Mostrar únicamente lo necesario.

La complejidad aparece cuando el usuario la necesita.

## 7. Search First

Todo debe poder encontrarse rápidamente.
